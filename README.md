# Проект обработки банковских операций (PytonProject1)

## Описание проекта
Этот проект представляет собой пакет утилит на языке Python для обработки данных банковских транзакций. Он позволяет фильтровать операции по их статусу, сортировать по дате (с корректной обработкой часовых поясов и стандартов ISO 8601), а также безопасно маскировать персональные данные клиентов (номера карт и счетов) перед отображением в интерфейсе.

## Функционал модуля `processing.py`
* **`filter_by_state(data, state)`**: Фильтрует список словарей транзакций по указанному статусу (например, `EXECUTED`, `CANCELED`).
* **`sort_by_date(data, reverse=True)`**: Надежно сортирует транзакции по дате, предварительно преобразуя строковые значения в объекты `datetime`. Защищен от пустых значений дат и некорректных форматов.

## Установка и настройка

Проект использует менеджер пакетов **Poetry**.

1. Клонируйте репозиторий:
   ```bash
   git clone <https://github.com/genrik270-design/Pythonproject1/tree/develop>
   cd PytonProject1
   ```

2. Установите зависимости виртуального окружения:
   ```bash
   poetry install
   ```

## Тестирование

В проекте используется библиотека `pytest` для автоматического тестирования модулей `masks`, `widget` и `processing`. Тесты проверяют корректность маскирования карт/счетов, форматирование дат, фильтрацию данных, а также устойчивость функций к некорректным или пустым входным данным.

Для запуска тестов выполните в терминале команду:
```bash
poetry run pytest -v
```

Вы также можете запустить сбор отчета о покрытии кода тестами (если в проекте настроен `pytest-cov`):
```bash
poetry run pytest --cov=src --cov-report=term-missing
```
