## Модуль `generators` (Работа с генераторами)

Модуль предназначен для эффективной обработки массивов транзакций с помощью итераторов и функций-генераторов. Использование ключевого слова `yield` позволяет обрабатывать неограниченные объёмы данных с минимальным потреблением оперативной памяти.

### Описание функций и примеры использования

Для начала работы импортируйте необходимые функции в свой рабочий файл:
```python
from src.generators import (
    filter_transactions_by_currency,
    transaction_descriptions,
    card_number_generator
)
```

#### 1. Фильтрация транзакций по валюте `filter_transactions_by_currency`
Принимает на вход итерируемый объект со словарями транзакций и строковый код валюты (например, `"USD"`, `"RUB"`). Возвращает по одной транзакции, соответствующей условию.

*Пример кода:*
```python
transactions = [
    {"id": 1, "operationAmount": {"currency": {"code": "RUB"}}, "description": "Перевод"},
    {"id": 2, "operationAmount": {"currency": {"code": "USD"}}, "description": "Покупка"}
]
```
rub_transactions = filter_transactions_by_currency(transactions, "RUB")
print(next(rub_transactions))  # Выведет первую транзакцию в рублях
```

#### 2. Получение описаний операций `transaction_descriptions`
Принимает список транзакций и поочередно возвращает текстовое описание (`description`) для каждой операции. Если описание отсутствует, возвращает заглушку `"Описание отсутствует"`.

*Пример кода:*
```python
transactions = [
    {"id": 1, "description": "Перевод организации"},
    {"id": 2}  # Без описания
]

descriptions = transaction_descriptions(transactions)
print(next(descriptions))  # "Перевод организации"
print(next(descriptions))  # "Описание отсутствует"
```

#### 3. Генератор номеров карт `card_number_generator`
Генерирует номера карт в заданном диапазоне (от `start` до `end` включительно), автоматически дополняя их нулями до 16 знаков и форматируя блоками по 4 цифры (`0000 0000 0000 0000`).

*Пример кода:*
```python
```
cards = card_number_generator(1, 2)
print(next(cards))  # "0000 0000 0000 0001"
print(next(cards))  # "0000 0000 0000 0002"
```

# Проект обработки банковских операций (PytonProject1)

## Описание проекта
Этот проект представляет собой пакет утилит на языке Python для обработки данных банковских транзакций. Он позволяет фильтровать операции по их статусу, сортировать по дате (с корректной обработкой часовых поясов и стандартов ISO 8601), а также безопасно маскировать персональные данные клиентов (номера карт и счетов) перед отображением в интерфейсе.

## Функционал модуля `processing.py`
* **`filter_by_state(data, state)`**: Фильтрует список словарей транзакций по указанному статусу (например, `EXECUTED`, `CANCELED`).
* **`sort_by_date(data, reverse=True)`**: Надежно сортирует транзакции по дате, предварительно преобразуя строковые значения в объекты `datetime`. Защищен от пустых значений дат и некорректных форматов.
